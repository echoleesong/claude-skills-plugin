# Claude 模型验证器 (claude-verifier)

## 描述
验证 Claude 模型是否为"纯血"官方模型，检测掺假、套壳、降级等问题的完整指南。

## 触发条件
当用户需要：
- 验证 Claude API/Key 的真实性
- 检测模型是否被降级（如用 4.5 冒充 4.6）
- 确认 Claude Code 是否为原版
- 测试第三方 API 服务是否可靠

### 完整验证清单

| 测试项 | 检测目标 | 优先级 |
|--------|----------|--------|
| 分词器测试 | 验证是否为 Claude 原生分词器 | 高 |
| count_tokens 接口 | 验证 API 完整性 | 高 |
| 知识库截止时间 | 判断具体模型版本 | 高 |
| 联网搜索测试 | 验证 CC 原生功能 | 中 |
| MCP 工具调用 | 验证工具链支持 | 中 |
| 提示词注入测试 | 验证系统提示词处理 | 中 |
| 温度参数测试 | 验证参数真实性 | 中 |
| Signature 验证 | 验证工具调用签名 | 低 |
| 新参数验证 | 检测 Opus 4.6 特有参数 | 低 |
